# Python

Python is a high-level, general-purpose programming language. It was created by
Guido van Rossum and first released in 1991. The language emphasises code
readability and uses significant indentation.

Python is dynamically typed and garbage-collected. It supports multiple
programming paradigms, including procedural, object-oriented, and functional
programming. The name "Python" was inspired by the British comedy group Monty
Python, not the snake.
